# CODING-SAMURAI-INTERNSHIP-TASK

Welcome to the Business Analytics Internship Tasks Repository! This repository contains all the tasks and projects completed during my business analytics internship at Coding Samurai.

## Projects

### 📊 Project 1: Sales Metrics Dashboard
A basic sales performance dashboard built in Power BI for a small retail store.

**Objective:** Create a sales performance dashboard to track key business metrics.

**Tasks:**
- Calculated basic sales metrics — total revenue, average order value, and total sales
- Visualized monthly sales trends using line and bar charts
- Identified the top 3 products by sales and visualized them in a bar chart

**Tools:** Power BI

**Files:**
- `Sales_dashboard.pbix`
- `product_sales_dataset_final.csv`

---

### 📈 Project 2: Monthly Sales Trends and Analysis
Analysis of monthly sales data to identify trends and regional performance.

**Objective:** Identify sales trends over time.

**Tasks:**
- Calculated monthly sales totals and displayed them in a line chart
- Identified the best and worst months for sales
- Analyzed sales by region using a bar chart

**Tools:** Python (Pandas, Matplotlib)

**Files:**
- `monthly_sales_analysis.ipynb`
- `product_sales_dataset_final.csv`

---

### 🔮 Project 3: Sales Predictions Using Basic Models
Predicting future sales by analyzing past trends using a machine learning model.

**Objective:** Predict future sales using trends.

**Tasks:**
- Performed trend analysis using a Random Forest model to project next month's sales
- Visualized actual vs predicted revenue using a line chart
- Suggested adjustments based on observed trends

**Tools:** Python (Pandas, Seaborn, Scikit-learn)

**Files:**
- `sales_predictions.ipynb`
- `product_sales_dataset_final.csv`
